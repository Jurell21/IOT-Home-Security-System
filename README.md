# IoT Home Security & Alarm System

## Project Overview
Designed and implemented a microcontroller-based home security system that detects motion and door activity and triggers automated alarm responses. The system simulates real-world smart home security workflows used in residential environments.

## Key Features
- Motion and door sensor integration  
- Automated alarm and visual alert responses  
- Event-driven logic to reduce false triggers  
- Modular design for scalability  

## Technologies Used
- Arduino  
- Digital I/O  
- Embedded C/C++  
- Sensors & alarms  

## Use Case
- Smart home security, residential monitoring systems, IoT infrastructure.
